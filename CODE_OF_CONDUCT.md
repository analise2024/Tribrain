# Code of Conduct

This project follows the Contributor Covenant Code of Conduct (v2.1). In short: be respectful, constructive, and inclusive.

## Our standards
We expect:
- Empathy and kindness
- Respect for different viewpoints and experiences
- Constructive feedback
- Responsible disclosure of security issues

We do not tolerate:
- Harassment, discrimination, or hateful conduct
- Threats or incitement of violence
- Sharing private information without consent

## Enforcement
If something goes wrong, open an issue labeled **conduct** or email the maintainer listed in `SECURITY.md`.
